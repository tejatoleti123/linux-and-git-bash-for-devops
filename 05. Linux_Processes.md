## Process Management Commands

`ps`  
Displays processes for current terminal session only.

`ps aux`  
Shows ALL processes with user, CPU%, MEM%, and command details.


`ps -ef`  
BSD-style process list showing parent/child PIDs and start time.

`top`  
Interactive real-time process monitor (press `q` to quit, `k` to kill).

`htop`  
Enhanced top with mouse support, tree view, and better layout.

`pgrep nginx`  
Finds PIDs by process name (returns multiple PIDs if found).

`pgrep -f "backup script"`  
Matches full command line patterns.

`pkill nginx`  
Kills all processes by name (graceful SIGTERM).

`pkill -9 nginx`  
Force kills processes by name (SIGKILL).

`kill 1234`  
Sends SIGTERM (graceful termination) to PID 1234.

`kill -9 1234`  
Force kills process with SIGKILL (last resort).

`kill -l`  
Lists all available signal numbers and names.

`killall nginx`  
Kills all processes named nginx (same as pkill).

`jobs`  
Lists background jobs in current shell session.


## Process Management Notes

**Pro Tips:**
- **Find + Kill:** `pkill nginx` or `kill $(pgrep nginx)`
- **High CPU:** `top`, press `Shift+P` to sort by CPU usage
- **Memory leaks:** `top`, press `Shift+M` for memory sort
