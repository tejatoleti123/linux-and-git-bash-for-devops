## Filesystem Commands

`pwd`  
Prints the current working directory, helping confirm where commands are being executed.

`ls`  
Lists files and directories in the current directory.

`ls -l`  
Lists files in long format, showing permissions, ownership, file size, and timestamps.

`ls -a`  
Lists all files, including hidden files (those starting with `.`).

`ls -la`  
Lists all files, including hidden ones, in long format with permissions and ownership details.

`cd <path>`  
Changes the current working directory to the specified path.

`cd ..`  
Moves one directory level up from the current directory.

`cd ~`  
Navigates to the current userâ€™s home directory.

`mkdir <directory>`  
Creates a new directory.

`mkdir -p <parent>/<child>`  
Creates nested directories safely; does not error if directories already exist.

`rmdir <directory>`  
Removes an empty directory only.

`touch <file>`  
Creates an empty file or updates the timestamp of an existing file.

`cp <source> <destination>`  
Copies a file from source to destination.

`cp -r <source_dir> <destination_dir>`  
Copies a directory and its contents recursively.

`mv <source> <destination>`  
Moves or renames files and directories.

`rm <file>`  
Deletes a file permanently.

`rm -r <directory>`  
Deletes a directory and its contents recursively.

`rm -rf <directory>`  
Forcefully deletes a directory and its contents without confirmation (use with caution).

`find . -name "*.log"`  
Searches for files by name recursively starting from the current directory.

`df -h`  
Displays disk space usage of mounted filesystems in human-readable format.

`du -sh <path>`  
Shows the total size of a file or directory in human-readable format.

`stat <file>`  
Displays detailed information about a file, including size, permissions, and timestamps.

`tree`  
Displays directories and files in a tree-like structure (may require installation).
