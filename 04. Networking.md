# Networking Commands

ip a  
Displays network interfaces and IP addresses.

ifconfig  
Shows network configuration (legacy command).

netstat -tulpn  
Lists listening ports and associated services.

ss -tulpn  
Displays active socket connections.

ping google.com  
Tests network connectivity.

curl -I https://example.com  
Fetches HTTP response headers.

wget URL  
Downloads files from the internet.
