## Package Management Commands

**YUM/DNF (RHEL/CentOS/AlmaLinux/Amazon Linux)**

`yum update`  
Refreshes package lists and upgrades all packages (interactive).

`yum update -y`  
Non-interactive full system update (scripting/CI/CD).

`yum install nginx`  
Installs nginx web server and dependencies.

`yum install nginx-1.20* -y`  
Installs specific version pattern without prompts.

`yum remove nginx`  
Removes package but keeps config files.

`yum autoremove`  
Removes unused dependencies (orphans).

`yum search mysql`  
Finds packages matching keyword in repo.

`yum list installed | grep php`  
Lists all installed PHP packages.

`yum info nginx`  
Shows package details: version, size, deps, description.

`yum clean all`  
Clears cache/metadata (fix repo errors).

`yum repolist enabled`  
Lists active repositories with package counts.

**APT (Ubuntu/Debian)**

`apt update`  
Downloads latest package lists (always first).

`apt upgrade -y`  
Upgrades packages keeping current config.

`apt install apache2`  
Installs Apache with dependencies.

`apt autoremove`  
Cleans orphaned packages.

`apt search postfix`  
Searches repository for packages.

## Package Management Notes

**Distro Quick Reference:**

- CentOS/RHEL 7:     yum
- RHEL 8+:           dnf  
- Ubuntu/Debian:     apt
- Amazon Linux 2:    yum
- Fedora:            dnf


**Troubleshooting:**

- No package:      yum repolist (check repos)
- GPG errors:      yum clean all  
- Dependency hell: yum swap old new
- Repo 404:        yum makecache
