# Package Management with yum

yum is the package manager used in Amazon Linux and RHEL-based systems. It is commonly used by DevOps engineers to install, update, and manage software on servers and cloud instances.


yum update  
Updates all installed packages to the latest available versions.

yum update -y  
Updates packages automatically without confirmation.

yum install package_name  
Installs a specified package.

yum install package_name -y  
Installs a package without prompting for confirmation.

yum remove package_name  
Removes an installed package.

yum search keyword  
Searches for packages matching a keyword.

yum list installed  
Lists all installed packages.

yum info package_name  
Displays detailed information about a package.

yum clean all  
Clears cached package data.
yum repolist  
Displays enabled yum repositories.
