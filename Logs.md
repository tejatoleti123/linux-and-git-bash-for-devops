# Log Analysis Commands

cat file.log  
Displays the entire log file.

less file.log  
Views log files page by page.

tail file.log  
Shows the last lines of a log file.

tail -f file.log  
Streams log entries in real time.

head file.log  
Displays the first lines of a log file.

grep ERROR file.log  
Searches for error entries in logs.

grep -i error file.log  
Performs a case-insensitive search in logs.
