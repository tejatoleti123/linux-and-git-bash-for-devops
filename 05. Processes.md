# Process Management

ps  
Displays running processes.

ps aux  
Shows detailed information about all running processes.

top  
Monitors processes and system resource usage in real time.

kill PID  
Terminates a process gracefully.

kill -9 PID  
Forcefully terminates a process.

pgrep process_name  
Finds the process ID by name.

pkill process_name  
Kills processes by name.
