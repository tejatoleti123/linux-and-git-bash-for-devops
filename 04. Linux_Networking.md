## Networking Commands

`ip a`  
Displays all network interfaces with their IP addresses, status (UP/DOWN), and MAC addresses.

`ip addr show`  
Alternative to `ip a` - shows detailed interface addresses and routing information.

`ifconfig`  
Legacy command showing network interfaces (deprecated; use `ip` instead).

`ip route`  
Displays the kernel routing table with default gateway and connected networks.

`ping -c 4 google.com`  
Tests connectivity to a host with 4 packets (add `-c` to limit count).

`traceroute google.com`  
Shows the route packets take to reach a destination (hop-by-hop).

`curl -I https://example.com`  
Fetches HTTP headers only to test web server connectivity and SSL.

`curl -v https://example.com`  
Verbose curl showing complete request/response handshake.

`wget -qO- https://example.com`  
Quiet wget that outputs content directly (no progress bar or files).

`nslookup google.com`  
Interactive DNS query tool (legacy but still widely used).

`hostname -I`  
Shows all configured IP addresses on the local machine.


## curl vs wget - Detailed Comparison

**curl** - Transfer tool for testing, debugging, APIs, and data transfer

**wget** - Download manager focused on robust file retrieval
